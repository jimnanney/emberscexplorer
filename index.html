Ember SC Explorer.

This application is a quick tutorial introduction to using Ember JS to consume a public API, Soundcloud's public API. Within the application we will build a sound search, a user browser, and a quick sound player using the sound cloud streaming api.

Each step along the way is a branch within the repository allowing you to easily roll back and move forward through the tutorial.

This will not be a ember cli project, but instead the ember appkit style project, as the initial group I'm presenting too has no access to npm and node.

Getting Started

To get started simply clone the repository and checkout branch step1.

In your shell (or Git Bash if on windows) clone the repo with the following commands.

$ git clone git://github.com/jnanney/emberscexplorer
$ git checkout -b origin/step1

This tutorial is MIT licensed so feel free to clone, modify or use it as you need. I would appreciate any feedback or contributions, and will respond quickly to any issues.

Enjoy!
